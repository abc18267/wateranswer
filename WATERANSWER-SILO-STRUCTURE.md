## INTERNAL LINKING RULES

1. Every page links to 3+ pages within its own silo
2. Every page links to 1-2 pages in other silos where relevant
3. Every contaminant page links to: its testing page + its treatment page + its worst-affected states
4. Every product review page links to: the contaminant it solves + the testing page + NSF certification explainer
5. Every state page links to: the contaminants common there + local testing labs + recommended treatment
6. Hub pages link to ALL their spokes. Spoke pages link back to hub + 2-3 sibling spokes.

## URL Rules

- Max 3 directory levels: `/silo/sub-silo/page/`
- No trailing slashes in content links (redirect trailing slash to non-trailing)
- Canonical URLs on every page

## On-Page SEO
- Title tag: Primary keyword — WaterAnswer.com (under 60 chars)
- H1: One per page, matches search intent
- H2s: Sub-topics, long-tail keyword targets
- FAQ schema: On every informational page (3-5 questions)
- Product schema: On every review/comparison page
- HowTo schema: On every treatment/guide page
- BreadcrumbList schema: On every page
- Internal links: Minimum 5 per page, contextual anchor text
- Images: Custom diagrams, comparison tables, process illustrations
- Alt text: Descriptive, keyword-relevant
